# Swiggy Sales Analysis

This project analyzes Swiggy Sales Analysis data using Python and Google Colab.

## Technologies Used
Python
Pandas
NumPy
Matplotlib
Google Colab

## Analysis Performed
Weekly Trend Analysis
Price Analysis
Category Analysis
Restaurant Performance
Rating Analysis

## Dataset
Swiggy data dataset containing information about restaurants, food categories, prices, ratings, and order dates.
